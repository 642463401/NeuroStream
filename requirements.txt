# NeuroStream 依赖清单(v0.2.0 — 方案 A 持续学习训练 + 未来 OpenAI 兼容 API)
#
# 严格依赖定义在 pyproject.toml 的 [project.optional-dependencies],本文件
# 是"一行装齐"的便利清单,等价于:
#   pip install -e ".[decoder,agent,gpu,dev]" + ijson + fastapi/uvicorn/openai
#
# 用法:
#   pip install -r requirements.txt
#
# 注意:torch 默认装的是 CPU 版,GPU 训练前必须先单独装 CUDA wheel。
# 不同显卡架构需要不同 CUDA 版本:
#   - RTX 30/40 系 (Ampere/Ada, sm_80-89)  → cu121 或 cu124 均可
#   - RTX 50 系   (Blackwell,    sm_120)   → 必须 cu128(cu121 会报
#                                            "no kernel image" 错误)
# 安装命令(任选其一,匹配你的显卡):
#   pip install torch --index-url https://download.pytorch.org/whl/cu128   # RTX 5060/5070/5080/5090
#   pip install torch --index-url https://download.pytorch.org/whl/cu124   # RTX 30/40 系
# 然后再 pip install -r requirements.txt(已装的 torch 不会被覆盖)
#
# 验证 GPU 可用:
#   python -c "import torch; print(torch.cuda.is_available(), torch.cuda.get_device_capability(0))"
